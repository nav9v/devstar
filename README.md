## Run Locally

Clone the project

```bash

git  clone  https://github.com/AraeneaCLI/devstar.git

```

Go to the project directory

```bash

cd  devstar

```

Install dependencies

```bash

npm  install

```

Start the server

```bash

npm  run  dev

```
